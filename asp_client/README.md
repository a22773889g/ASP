# Installation
1. Clone the project 
```
git clone https://github.com/a22773889g/ASP.git
```
2. Go to the client directory 
```
cd asp_client/
```
3. Install dependencies
```
yarn install
    or 
npm install
```

4. Open the service 
```
yarn start 
   or 
npm start
```
